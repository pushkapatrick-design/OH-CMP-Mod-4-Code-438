Code 438

Code 438 - CIP Not Transferred to New Pole

{/* Image below title */}

When to Use Code 438

Equipment transfer after pole change
Use Code 438 when CIP needs to transfer their equipment to a new pole after SDG&E has changed out a pole. This ensures proper tracking of equipment relocation.

Default Settings in Geocall

Automatically assigned severity
The condition will default to 'Compliance' Risk and 'Low' Severity in Geocall. These automatic settings ensure consistent categorization across all Code 438 reports.

Required Documentation

CIP name & additional details
Always provide the CIP name and any additional details about the pole change and equipment transfer. Clear documentation helps coordinate the transfer work.

Pole Change Context

SDG&E initiated changes
This code specifically applies when SDG&E has changed out a pole, not for other types of pole work. The CIP equipment must be transferred to the new pole location.
